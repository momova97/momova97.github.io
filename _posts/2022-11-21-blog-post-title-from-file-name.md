# Introduction to Data Visualization with Matplotlib
You learn about plot structures and the functionalities employed during this course. You learn how to label the axes and how to add data to them. Additionally, you started creating subplot grids using the subplot(rows, columns) method.
By importing time series data into the plot function and learning how to annotate a time-series graph, we can easily construct time series plots in Python.
You generated stacked bar charts by using fundamental Python functions like ax.bar(). Using the functions ax.boxplot() and ax.scatter(), I learn how to make boxplots and scatterplots.
You learn how to create figures to share in the last chapter.
You become adept at utilizing the plt.style.use() function to change styles. The visualization automation in Python using a "for" loop and repeating it after a specified condition and change was the greatest part of this chapter. Finally, you understand how to save a file with a name and specified dpi.
